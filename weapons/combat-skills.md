# 武器スキル

> 📌 情報源: [FextraLife Wiki](https://soulmask.wiki.fextralife.com/Soulmask_Wiki) / [games.gg/soulmask](https://games.gg/soulmask/)

## 武器スキルシステムの概要

各武器クラスは使用によって熟練度が上がり、熟練度30/60/90/120の4段階でスキルが解放されます。9クラス × 約9スキル = 全88種。

スキルは大きく4カテゴリに分類されます：

| カテゴリ | 説明 |
|---------|------|
| 接近戦スキル | 高ダメージ・範囲攻撃 |
| 逃げスキル（I-frame付き） | 発動中に無敵判定あり。回避しながら攻撃できる |
| スーパーアーマースキル | 被ダメージでモーションがキャンセルされない |
| 状態異常付与スキル | 出血・スタン・スタミナ削りなど |

## 各武器クラスの代表スキル

| 武器クラス | スキル名 | 効果 |
|-----------|---------|------|
| **Blade**（片手剣） | Tiger Strike | ダッシュ突進＋I-frameで無敵回避しながら攻撃 |
| **Spear**（槍） | Ultimate Throw | 槍を投擲してノックダウン |
| **Bow**（弓） | Back Jump Shot | バックジャンプしながら複数の矢を発射 |
| **Hammer**（大槌） | Solar Protection | スーパーアーマー中に50%ダメージ軽減 |
| **Shield**（盾） | Shield Counter | 攻撃を受け流してからカウンター攻撃 |
| **Gauntlets**（拳鎧） | Revolving Kick | 敵のスタミナを大量に削るキック連打 |
| **Dual Blades**（双剣） | Lightning Speed | 6連続前方突きで出血スタック蓄積 |
| **Greatsword**（大剣） | Quadru-Tornado Slash | 4連回転斬り＋スーパーアーマー。高DPS |
| **Spiked Whip**（棘鞭） | Wind-breaking Whip | AGI/STR比率によってモーション変化する前方打撃 |

## スキルスロット

戦闘中に発動できるスキルをホットバーのスロットに事前登録します。武器クラスごとにスキルセットを切り替えながら使用します。

## スキルの活用指針

**I-frameスキル（Blade: Tiger Strike等）**  
ボス攻撃の回避タイミングに合わせて発動すると、回避しながらダメージを与えられます。

**スーパーアーマースキル（Hammer: Solar Protection等）**  
強敵の怯みを無視して強引に攻撃を通したい場面で有効。

**スタミナ削りスキル（Gauntlets: Revolving Kick等）**  
スタミナを削り切ると敵が一時的に行動不能になります。強敵・ボスに対して有効。
