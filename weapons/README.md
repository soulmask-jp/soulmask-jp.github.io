# 武器・戦闘

> 📌 情報源: [Steam ストアページ](https://store.steampowered.com/app/2646460/Soulmask/) / [FextraLife Wiki](https://soulmask.wiki.fextralife.com/Soulmask_Wiki)

## 武器・戦闘システムの概要

Soulmaskの戦闘は**9種の武器クラス**と**88の武器スキル**を基盤とした多彩なシステムです。さらに「神話システム（Mythology System）」を活用することで戦況を大きく変えることができます。

## 武器クラス一覧

Soulmaskには9種の武器クラスがあります。詳細は[武器種一覧](weapon-types.md)を参照。

| 武器クラス | スタイル |
|-----------|---------|
| Blade（片手剣） | 機動型・I-frame回避 |
| Spear（槍） | バランス型・長リーチ |
| Bow（弓） | 遠距離型 |
| Hammer（大槌） | 重量型・超火力 |
| Shield（盾） | 防御反撃型 |
| Gauntlets（拳鎧） | 格闘型・スタミナ削り |
| Dual Blades（双剣） | 高速型・出血特化 |
| Greatsword（大剣） | 両手高火力型 |
| Spiked Whip（棘鞭） | 遠近両用型（1.0追加） |

## 武器スキル（88種）

各武器クラスには固有の武器スキルがあり、戦闘中に発動することで強力な攻撃や特殊効果をもたらします。スキルの組み合わせによってプレイスタイルを幅広くカスタマイズできます。

詳細は[武器スキル](combat-skills.md)を参照。

## 神話システム（Mythology System）

マスクの「神の象徴（Mimicry Ascension）」として体現することで発動できる強化形態です。Sunstone（サンストーン）を消費して発動し、強力な戦闘アビリティが使用可能になります。発動できるマスク・具体的な効果は[マスクシステム](../mask-system.md)を参照。

## 戦闘の基本

- **回避**: スタミナを消費してダッシュ回避
- **ガード**: Spear（槍）・Shield（盾）など一部武器でブロック可能
- **スキル**: 武器固有スキルをスロットに割り当てて発動
- **部族員との連携**: 部族の戦闘員と連携した集団戦が有効
