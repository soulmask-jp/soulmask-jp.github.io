# 序盤攻略

> 📌 情報源: [myfavsteamgame.com](https://myfavsteamgame.com/soulmask-early-game-guide1/) / [enjpgamer.com](https://enjpgamer.com/soulmask-guide-1/) / [note.com/erystia](https://note.com/erystia/n/n06a4c4597e66)

## ゲーム開始直後の流れ

### Step 1: チュートリアルをクリアする
1.0で追加された独立チュートリアルモードで基本操作（移動・採集・クラフト・戦闘）を習得しましょう。

### Step 2: スターターマスクを選ぶ
ゲーム開始時に3種のマスクから1つを選びます。

| マスク | 推奨対象 |
|--------|---------|
| **Civilization** | 初心者・万能型（被攻撃時HP回復で生存しやすい） |
| **Conquest** | ステルス・アサシン志向 |
| **Rich** | 弓術・資源管理重視 |

→ 詳細は[マスクシステム](mask-system.md)を参照。

### Step 3: 石器・基本装備を揃える
原始工具（斧・ツルハシ・ナイフ）を焚き火（Campfire）でクラフトして採集効率を確保します。テクノロジーツリー（`T`キー）の序盤ノードを優先的に解放しましょう。

### Step 4: 最初の蛮族を3〜6人勧誘する

**Soulmask最大の特徴**は蛮族の勧誘です。序盤から積極的に行いましょう。

**目標構成:**
| 役割 | 人数 | 才能の目安 |
|------|------|-----------|
| 戦闘員（近接） | 1〜2人 | 武器熟練・体力系 |
| 戦闘員（遠距離） | 1人 | 弓・遠距離系 |
| 採取・クラフト員 | 1〜2人 | 採掘・クラフト系 |

→ 詳細は[蛮族の勧誘・才能](tribe/recruitment.md)を参照。

### Step 5: 拠点を構築・自動化ラインを立ち上げる
蛮族に役割を割り当てながら農業・採掘の自動化ラインを整備します。倉庫は製造台・農地の近くに配置することで搬送効率が上がります。

→ 詳細は[自動化・ワークライン](tribe/automation.md)・[建築ガイド](crafting/building.md)を参照。

### Step 6: Lorestonesを集めながら探索する
マップ各地にある「Lorestone（古代石板）」を読むと技術ポイントが獲得でき、新しいレシピ・製造台がアンロックされます。移動中に積極的に収集しましょう。

## 序盤の優先事項

| 優先度 | やること |
|--------|---------|
| ★★★ | 基本採集ツールの作成（Campfire → Workbench） |
| ★★★ | 最初の蛮族3人の勧誘 |
| ★★★ | 安全な拠点の確保（壁・柵で防衛） |
| ★★ | 農業ラインの立ち上げ（食料安定化） |
| ★★ | Lorestonesを集めて技術ポイント獲得 |
| ★ | マスクの強化（Holy Ruin進行） |

## ゲームモード別の序盤方針

| モード | 序盤の重点 |
|--------|----------|
| **サバイバルモード** | バランスよく全システムを進める |
| **トライブモード** | 部族員の増員・農業ライン構築を先行 |
| **ウォーリアーモード** | 戦闘装備を優先。管理業務は自動化に任せる |

→ [ゲームモード](game-modes.md)参照。

## Tier 1〜2エリアの攻略

序盤は **Eastern Rainforest（Tier 1, Lv1〜10）** からスタートし、Western/Northern Rainforest（Tier 2, Lv11〜20）へ進出します。自分のレベルより大幅に高いTierには入らないよう注意。

→ [本編マップ](maps/main-map.md)でエリアの詳細を確認。
