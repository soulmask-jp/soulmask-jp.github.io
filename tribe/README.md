# 部族・人材育成

> 📌 情報源: [Steam ストアページ](https://store.steampowered.com/app/2646460/Soulmask/) / [wicurio.com/soulmask/](https://www.wicurio.com/soulmask/index.php?%E9%83%A8%E6%97%8F=)

部族管理はSoulmaskの**最大の差別化要素**です。敵の蛮族を洗脳・勧誘して自分の部族員にし、様々な役割に配置することで高度な自動化生産体制を構築できます。

## 部族システムの概要

プレイヤーはマスクの能力を使って蛮族の意識に干渉し、味方に引き入れます。勧誘した蛮族は以下の役割に配置可能です。

- **戦闘員**: 拠点警備・野外戦闘に参加
- **採掘員**: 鉱石・素材の自動採集
- **農業員**: 農場の種植え・収穫を自動化
- **クラフト員**: 製造台での自動クラフト
- **料理人**: 食料の自動調理
- **建設員**: 建築補助（情報収集中）

## 才能システム（Talent System）

蛮族は**871種以上のユニーク才能**を持ちます。才能は蛮族ごとにランダムに決まり、同じ才能の組み合わせを持つ蛮族はほぼ存在しません。

才能の例：
- 採掘速度アップ
- 農業効率アップ
- 特定武器の熟練度アップ
- 体力・防御力アップ
- クラフト速度アップ

才能に合った役割に配置することで、生産効率が大幅に向上します。
