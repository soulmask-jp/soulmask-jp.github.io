# 蛮族の勧誘・才能

> 📌 情報源: [games.gg](https://games.gg/soulmask/guides/soulmask-tribesmen-guide/) / [gamerant.com](https://gamerant.com/soulmask-how-to-recruit-more-tribe-members/) / [clawsomegamer.com](https://clawsomegamer.com/how-to-recruit-tribesmen-in-soulmask/)

## 勧誘の流れ（2ステップ）

蛮族の勧誘は「Deter（制圧）→ Recruit（勧誘）」の2段階で完了します。特別な縄や道具は不要です。

### Step 1: Deter（制圧）

1. **対象を見つける**: Barbarian（蛮族）は遺跡に、Outcast（追放者）は兵舎（Barracks）に出現します
2. **単独を狙う**: 集団戦は危険なので、単独の個体を引き離してから戦う
3. **HP20%以下まで削る**: 殺さないよう注意しながら攻撃
4. **Deter実行**: HP20%以下になるとプロンプトが表示されるので `E` キーを押す
5. 対象が**気絶状態**になり、Recognition（信頼度）が徐々に蓄積し始める

### Step 2: Recognition が500になるまで待つ

気絶中の蛮族はそのままにしておいてもRecognitionが上がりますが、以下のケアで生存確率が上がります：

- インベントリに**包帯**を入れる → 自動で体力回復
- インベントリに**スープ等の液体食料**を入れる → 状態維持

気絶中の蛮族の**移送方法**：
- 肩に担いで運ぶ
- ラマの拡張座席に乗せる

### Step 3: Recruit（勧誘）

Recognition が **500** に達したら勧誘オプションが出現します。`E` キーで正式に部族員として加えましょう。

## 部族員の上限人数

| 状態 | 上限人数 |
|------|---------|
| 初期（Control ノードのみ） | 3人 |
| Connection Enhancement を最大アップグレード | **18人** |

Connection Enhancement はテクノロジーツリー（`T`キー）から研究できます。

## 才能（Talent）カテゴリ

勧誘した部族員は複数の才能カテゴリでボーナスを持ちます：

| カテゴリ | 内容 |
|---------|------|
| 先天的利点（Innate Advantage） | 生まれ持った能力強化（採掘・戦闘・農業など） |
| 先天的欠陥（Innate Flaw） | 特定の能力の弱化 |
| 称号ボーナス（Title Bonus） | 特定の出身部族・キャリアによるボーナス |
| 部族固有才能（Tribe Talent） | 所属していた部族系統の特性 |
| レベルアップ才能 | 10レベルごとに新たな利点を獲得。5レベルごとに欠陥が消える可能性あり |

## 部族員の強化

- **装備を与える**: 武器・防具を持たせることで戦闘力向上
- **職業ボーナス**: 役割に就いた時間が長いほど熟練度上限が上昇し、効率が改善する
- **Training Ground**: 1.0で追加。部族員間でスキルを継承できる
- **役割を固定する**: 安定した役割に継続的に就かせることで複合ボーナスが蓄積する

## 才能に合った役割配置

採掘効率の高い才能を持つ蛮族を採掘員に、農業適性の高い蛮族を農業員に配置することで、生産効率が大幅に向上します（[自動化・ワークライン](automation.md)参照）。
