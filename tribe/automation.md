# 自動化・ワークライン

> 📌 情報源: [Steam ストアページ](https://store.steampowered.com/app/2646460/Soulmask/) / [soulmaskshiftingsands.wiki](https://soulmaskshiftingsands.wiki/)

## 自動化システムの概要

Soulmaskでは部族員に役割を与えることで、採掘・農業・クラフト・料理などの作業を**自動化**できます。適切に整備されたワークラインにより、プレイヤーが離席中も生産が継続します。

## 各ワークラインの設定方法

### 農業ライン
1. 農地（畑）を建設
2. 農業員（農業才能を持つ部族員）を農地に配置
3. 種を倉庫または農地にセット
4. 農業員が自動的に種植え→水やり→収穫を行う

農業については[クッキング・農業・畜産](../cooking-farming.md)も参照。

### 採掘・素材収集ライン
1. 採掘ポイント近くに採掘員を配置
2. 採掘員の才能（採掘効率）を確認して配置する
3. 採掘した素材は倉庫へ自動搬入（倉庫との連携設定が必要な場合あり）

### クラフトライン
1. 製造台（鍛冶炉・裁縫台等）を建設
2. クラフト員を製造台に配置
3. クラフトキュー（クラフト待ち行列）に作成したいアイテムを登録
4. クラフト員が素材があり次第、自動でクラフトを実行

### 料理ライン
1. 炊事場・調理台を建設
2. 料理才能を持つ部族員を料理人として配置
3. 食材と調理レシピを設定
4. 自動で料理を生産

詳細は[クッキング・農業・畜産](../cooking-farming.md)を参照。

## 効率的なワークライン設計のコツ

**倉庫の配置を最適化する**  
製造台・農地・採掘ポイントに近い場所に倉庫を配置することで、素材の搬送距離を短縮できます。

**才能に合った役割配置**  
採掘効率+50%の才能を持つ部族員を採掘員にするなど、才能に沿った配置が生産効率に直結します（[蛮族の勧誘・才能](recruitment.md)参照）。

**士気（Morale）管理（DLC）**  
Shifting Sands DLCでは士気管理システムが強化されます。部族員の士気が低いと作業効率が下がるため、食事・休暇・娯楽設備などで士気を維持しましょう（詳細は情報収集中）。

## Shifting Sands DLCでの変化

DLCではワークラインシステムが拡張され、エアシップ上での生産ラインも構築可能になります。役割配置・士気管理・生産ライン自動化がより重要になります。

詳細は[エアシップ（空中要塞）](../shifting-sands/airship.md)を参照。
