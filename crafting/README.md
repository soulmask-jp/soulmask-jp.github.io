# クラフト・建築

> 📌 情報源: [Steam ストアページ](https://store.steampowered.com/app/2646460/Soulmask/) / [FextraLife Wiki](https://soulmask.wiki.fextralife.com/Soulmask_Wiki)

## クラフトシステムの概要

Soulmaskのクラフトシステムは、素材を集めてツール・武器・防具・建築素材などを作成する標準的なサバイバルクラフト形式です。ただし、部族員にクラフト役割を与えることで**自動クラフト**が可能になる点が特徴です。

## クラフトの流れ

1. **素材採集**: フィールドで木・石・鉱石・植物などを採集
2. **製造台を建設**: 石器作業台→青銅炉→など段階的に解放
3. **クラフト実行**: 製造台でレシピを選択して実行
4. **自動化**: クラフト員を配置してキューに登録 → 自動生産（[自動化](../tribe/automation.md)参照）

## 技術進歩の段階

Soulmaskは石器時代→青銅器時代→鉄器時代→さらに高度な文明へと技術進歩します（詳細は情報収集中）。

| 時代 | 主な素材 |
|------|---------|
| 石器時代 | 石・木・骨 |
| 青銅器時代 | 銅・錫・青銅 |
| 鉄器時代 | 鉄鉱石・鋼鉄 |
| 上位段階 | 情報収集中 |
