# クラフト・建築

> 📌 情報源: [Steam ストアページ](https://store.steampowered.com/app/2646460/Soulmask/) / [FextraLife Wiki](https://soulmask.wiki.fextralife.com/Soulmask_Wiki)

## クラフトシステムの概要

Soulmaskのクラフトは、素材を集めてツール・武器・防具・建築素材などを作る標準的なサバイバルクラフト形式です。部族員にクラフト役割を与えることで**自動クラフト**が可能になる点が特徴です。

## クラフトの流れ

1. **素材採集**: フィールドで木・石・鉱石・植物などを採集
2. **製造台を建設**: Campfire（焚き火）→ Workbench（作業台）→ Furnace（炉）など段階的に解放
3. **クラフト実行**: 製造台でレシピを選択して実行
4. **自動化**: クラフト員を配置してキューに登録 → 自動生産（[自動化](../tribe/automation.md)参照）

## 技術の進歩段階

| 時代 | 主な素材 | 解放内容 |
|------|---------|---------|
| **石器時代（Stone Age）** | 石・木・骨 | 基本ツール・初期建材・焚き火 |
| **青銅器時代（Bronze Age）** | 銅・錫・青銅インゴット | 農業施設・穀倉・高度な建材・武器 |
| **鉄器時代（Iron Age）** | 鉄鉱石・鋼鉄インゴット | 上位武器・防具・製造台・石壁 |
| **上位段階** | 希少金属・特殊素材 | エンドゲーム装備・高度な自動化設備 |

青銅器時代から**穀倉・井戸・雨水収集器**が解放され、農業自動化が本格的に可能になります。
