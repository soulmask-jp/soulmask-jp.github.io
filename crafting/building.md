## 建築システムの概要

Soulmaskの建築システムは拠点の設計・拡張・防衛ラインの構築を担う核心システムです。部族の自動化ラインを効率的に運用するためのレイアウト計画が重要です。

## 建築の基本

1. **素材を用意する**: 木材・石材・金属など
2. **建築モードに切り替える**: 専用UIから建設したい構造物を選択
3. **設置場所を決定**: グリッド配置に沿って配置
4. **建設する**: 素材を消費して完成

## 建築カテゴリ

| カテゴリ | 具体例 |
|---------|--------|
| 構造物 | 壁・床・屋根・柱・ゲート |
| 防衛設備 | クロスボウタワー・攻城バリスタ・スパイクトラップ・石壁 |
| 生産設備 | 製造台・農地・牧場・倉庫・穀倉 |
| 居住設備 | ベッド・テーブル（士気向上） |
| 特殊施設 | Training Ground・繁殖農場 |

## 防衛設計

### 防衛施設の種類

| 施設 | 効果 |
|------|------|
| **クロスボウタワー（Crossbow Tower）** | 遠距離から自動で敵を攻撃する守備砲台 |
| **攻城バリスタ（Siege Ballista）** | より強力な遠距離防衛砲台。大型の敵や集団に有効 |
| **スパイクトラップ（Spike Trap）** | 木壁の前に設置して接近した敵にダメージ |
| **石壁（Stone City Wall）** | 木壁より耐久力が大幅に高い。資源コストは大きい |
| **パトロールルート（Patrol Route Checkpoint）** | 部族戦闘員の巡回ルートを設定。拠点外周を定期巡回させて侵入を早期発見 |

### 防衛の基本戦略

1. **多層防衛**: 外周に石壁・その内側にクロスボウタワー・入口にスパイクトラップを配置
2. **パトロール設定**: 武装した部族員に巡回ルートを割り当てて常時警戒
3. **弱点を補強**: 侵入経路になりやすい角・裏口を重点的に強化

### 侵攻（Invasion）への対備

Soulmaskでは定期的に敵が拠点へ侵攻してきます（Invasion Mechanics）。事前準備が不十分だと拠点を破壊されます。

- 防衛タワーの弾薬（矢・ボルト）を常に補充
- 部族員の装備を整えておく
- 弱点箇所を定期的に確認して補修

## 拠点設計のポイント

### 動線を最適化する
採掘ポイント → 倉庫 → 製造台 → 倉庫の流れが短くなるよう配置することで部族員の搬送効率が上がります。

### 農地は水源の近くに
農地は**川・池・井戸**に近い場所ほど農業自動化の効率が上がります。青銅器時代以降は雨水収集器や井戸を農地そばに建設しましょう。

### 穀倉を農地の近くに設置
部族員の農業自動化には穀倉が必要です。穀倉と農地の距離が短いほど搬送時間が短縮されます。
