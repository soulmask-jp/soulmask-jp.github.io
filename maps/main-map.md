# 本編マップ

> 📌 情報源: [FextraLife Wiki](https://soulmask.wiki.fextralife.com/Soulmask_Wiki) / [upgame.jp/smask/](https://upgame.jp/smask/index.php)

## マップ概要

本編マップの名称は**Cloud Mist Forest（クラウドミストフォレスト）**。**26リージョンを12ティア**に区分した広大なオープンワールドです。

全体構造は「南のジャングル → 草原 → 火山 → 北の雪山」という南北グラデーション。序盤は南のジャングルからスタートし、レベルを上げながら北の高難度エリアへ進出します。

## 探索のコツ

**ティアを意識して進む**  
自分のレベルより大幅に高いティアに踏み込むと一撃死する危険があります。序盤はTier 1〜2を安全に制圧してから次のエリアへ進みましょう。

**Lorestonesを集める**  
マップ各地にある「Lorestone（古代石板）」を読むと技術ポイントが獲得できます。移動中に積極的に収集しましょう（[テクノロジーツリー](../crafting/recipes.md)参照）。

**高所から俯瞰する**  
山や高台から周囲を見渡すと、蛮族の集落・ダンジョン・資源地帯の位置を把握しやすくなります。

**マーカーを活用する**  
有望な採掘スポット・蛮族集落・ダンジョン入口にマーカーを付けておくと後の探索が効率化されます。
