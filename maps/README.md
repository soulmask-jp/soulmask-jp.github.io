## マップ構成

Soulmaskには本編マップとDLCのエジプトマップ（Shifting Sands）の2つの大きなマップがあります。

| マップ | 概要 |
|--------|------|
| 本編マップ | 古代文明の大陸。多様なバイオームを持つ広大なオープンワールド |
| エジプトマップ（DLC） | 本編と同サイズの砂漠・ナイル川地帯。Shifting Sands DLC追加 |

## 重要ポイント

- **蛮族の集落**: 勧誘候補の蛮族が多数生息。集落を攻略して部族員を増やす
- **遺跡・ダンジョン**: マスクや強力な装備が眠る遺跡が各地に存在（[ボス攻略](../bosses.md)参照）
- **資源地帯**: 鉱石・特殊素材の採集に適した場所
