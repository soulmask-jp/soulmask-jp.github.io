# マップ概要

> 📌 情報源: [FextraLife Wiki](https://soulmask.wiki.fextralife.com/Soulmask_Wiki) / [saraserenity.net](https://saraserenity.net/soulmask/map/)

## マップ構成

| マップ | 名称 | 概要 |
|--------|------|------|
| **本編マップ** | Cloud Mist Forest（クラウドミストフォレスト） | 26リージョン・12ティア構成。南のジャングルから北の雪山まで多様なバイオーム |
| **エジプトマップ（DLC）** | Shifting Sands | 本編と同サイズの砂漠・ナイル川地帯。Shifting Sands DLC追加 |

## 本編マップ（Cloud Mist Forest）のバイオーム

全体構造は「**南のジャングル → 草原 → 火山 → 北の雪山**」という南北グラデーション。

| ティア | レベル帯 | 主なバイオーム |
|-------|---------|-------------|
| Tier 1 | Lv 1–10 | Eastern Rainforest（ジャングル）— スタート地点 |
| Tier 2 | Lv 11–20 | Western / Northern Rainforest |
| Tier 3–4 | Lv 16–30 | Mangrove・River Valley・Rift Valley・Rocky Bottomland |
| Tier 5–6 | Lv 26–40 | Wetland・Lakeside Forest・Table Mountain・Plateau Woodland |
| Tier 7 | Lv 36–45 | Great Prairie（大草原）・Giant Wood Forest（巨木の森） |
| Tier 8–9 | Lv 41–50 | Wasteland（荒廃地）・Volcanic Forest（火山林） |
| Tier 10 | Lv 46–55 | Scorching Volcano（灼熱火山） |
| Tier 11–12 | Lv 51–65 | Frost Leaf Grove・Dark Forest・Alpine Land・Frost Canyon（雪山） |

→ 詳細は[本編マップ](main-map.md)を参照。

## 重要ポイント

- **蛮族の集落**: 勧誘候補の蛮族が多数生息。集落を攻略して部族員を増やす（[蛮族の勧誘](../tribe/recruitment.md)参照）
- **遺跡・ダンジョン**: マスクや強力な装備が眠る遺跡が各地に点在（[ボス攻略](../bosses.md)参照）
- **Lorestone（古代石板）**: 読むことで技術ポイントを獲得（[テクノロジーツリー](../crafting/recipes.md)参照）
- **資源地帯**: ティアが上がるほど高品質な鉱石・素材が産出
