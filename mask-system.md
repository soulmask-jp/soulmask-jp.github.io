# マスクシステム

> 📌 情報源: [Steam ストアページ](https://store.steampowered.com/app/2646460/Soulmask/) / [FextraLife Wiki](https://soulmask.wiki.fextralife.com/Soulmask_Wiki)

マスクシステムはSoulmaskの**中核メカニクス**です。装備するマスクは古代文明の遺物として固有の能力を持ち、「意識転送（Mimicry）」によって他のキャラクターを直接操作することもできます。

## マスクの基本機能

### 意識転送（Consciousness Transfer / Mimicry）
マスクを通じて部族員の身体を操作できます。

- 前線の戦士として直接戦闘に参加
- 偵察員として地形探索
- 職人として精密作業
- プレイヤーキャラクターに自由に戻れる

### ミミクリーアセンション（Mimicry Ascension）
一部マスクはゲーム内通貨「Sunstone（サンストーン）」を消費して「神の象徴として体現」する強化形態に入れます。1.0では Rich / Torch of Eternity / Shadow Walker / Tactical Guidance の4マスクのミミクリースキルが大幅に再設計・強化されました。

### マスクの強化
マスクは「Holy Ruin（ホーリールイン）」のレベルアップや、ダンジョン内での修理・強化で能力を解放します。

## 本編マスク一覧（10種）

### スターターマスク（ゲーム開始時に3択）

| マスク名 | 主要アビリティ | 特性 |
|---------|-------------|------|
| **Civilization（シビライゼーション）** | Healer Mode（被攻撃時に体力回復）、Critical Barrier（頭部ダメージ-10%）、Stamina Surge（最大スタミナ増） | スタミナ・防御型。継戦能力が高い |
| **Conquest（コンクエスト）** | Slayer Mode（攻撃ごとに1.5%ライフスティール）、Hidden Assault（背後攻撃ダメージ増）、Noise Absorption（騒音軽減） | ステルス・戦闘型。アサシンスタイルに適している |
| **Rich（リッチ）** | 弓ダメージ強化・資源管理強化 | 弓術・ユーティリティ型。採集や遠距離戦に有利 |

### 後半入手マスク（7種）

Holy Ruinのレベル進行やダンジョン攻略で入手できます。

| マスク名 | 入手条件 | 主要アビリティ |
|---------|---------|-------------|
| **Torch of Eternity** | Holy Ruin Lv30 | 瀕死時に生存維持するアビリティ。高難度コンテンツに向く |
| **Wilderness Mark** | Holy Ruin Lv40 | 弓術・ハンティング強化 |
| **Tactical Guidance** | Holy Ruin Lv50 | Ionic Armor（部族員全員に電磁甲冑付与）。部族バフ特化 |
| **Shadow Walker** | Holy Ruin Lv55 | Optical Mimicry（透明化）、背後攻撃ダメージ+20%。ステルス特化 |
| **Explorer** | Holy Ruin Lv60 | Electromagnetic Barrier（エネルギー系攻撃耐性）、環境ダメージ完全防御。探索・対魔法型 |
| **Ironblood Guard** | ダンジョン修理 | Muscle Revival（スタミナ持続回復）、Vigilance Mode（近接ダメージ20%反射） |
| **Nature Enlightenment** | ダンジョン修理 | Nature's Heart（野獣召喚支援）、騎乗時ダメージ-30%。騎乗・ペット特化 |

## マスク選択の指針

| プレイスタイル | 推奨マスク |
|-------------|-----------|
| 序盤・万能型 | **Civilization**（継戦能力が高く初心者向き） |
| ステルス・アサシン | **Conquest** or **Shadow Walker** |
| 弓・遠距離 | **Rich** or **Wilderness Mark** |
| 部族員バフ重視 | **Tactical Guidance** |
| 探索・環境耐性 | **Explorer** |
| 騎乗・ペット利用 | **Nature Enlightenment** |
| 高難度ボス耐久 | **Torch of Eternity** or **Ironblood Guard** |

## 神の像（Statue of God）システム

マップ各地に存在する神の像に祈りやアイテムを奉納することで各種バフを獲得できます。戦闘中は部族員が神像周辺に集まり守護を祈る仕様もあります。
