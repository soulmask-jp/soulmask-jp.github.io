# マルチプレイ

> 📌 情報源: [primagames.com](https://primagames.com/gaming/soulmask-multiplayer-guide-how-to-join-and-invite-friends) / [gamertweak.com](https://gamertweak.com/how-to-join-friends-in-soulmask/) / [escapistmagazine.com](https://www.escapistmagazine.com/does-soulmask-have-multiplayer/)

## マルチプレイの種類

| モード | 概要 |
|--------|------|
| オンラインCo-op | Steam経由でフレンドを招待（最大10人程度） |
| 招待コード参加 | ゲーム開始時に表示されるコードを共有 |
| 部族名参加 | ホストが部族を作成して名前を共有、フレンドが検索して参加 |
| 公式サーバー | 最大**70人**対応の公式サーバーへ参加 |
| 専用サーバー | プレイヤーが独自に立てるサーバー（最大70人） |
| LANプレイ | 同一ネットワーク内での協力プレイ |

## 最大プレイヤー数

- **公式サーバー / 専用サーバー**: 最大 **70人**
- **Co-op（ホストセッション）**: Steamから最大 **10人**程度招待可能
- ホストのセッションの場合、ホストがオフラインになると他のプレイヤーもプレイ不可になります

## フレンドとの接続方法

### 方法1: 招待コード（最も簡単）
1. ゲームを起動して新規セッションを作成
2. セッション開始時に表示される**招待コード（Invitation Code）**をフレンドに共有
3. フレンドはメニューからコードを入力して参加

### 方法2: Steam招待
1. Steamのフレンドリストからフレンドを選択
2. ゲームに招待を送る
3. フレンドが招待を承認して参加

### 方法3: 部族名参加
1. ホストが `J` キーで部族を作成し、**部族名**をフレンドに伝える
2. フレンドが部族メニューで部族名を検索して参加申請
3. ホストが申請を承認

### 方法4: IP直接接続
サーバーのIPアドレスを直接入力して接続（専用サーバー向け）。

## マルチプレイの特徴

### Co-opの利点
- 部族員の確保・拠点建築が効率化
- ボス攻略の難易度が大幅に低下
- 役割分担（戦闘特化・クラフト特化・農業特化など）による効率化

### PvP要素
- Shifting Sands DLCではエアシップを使ったPvP空中戦闘が追加（[エアシップ](shifting-sands/airship.md)参照）
- サーバー設定でPvP/PvEを切り替え可能

## サーバーのカスタマイズ設定（200以上）

| カテゴリ | 設定例 |
|---------|-------|
| リソース | 素材の採集量・ドロップ率 |
| 戦闘 | 与ダメ・受ダメ倍率 |
| 部族管理 | 部族員の採用難易度 |
| 環境 | デイサイクルの速度・天候頻度 |
| PvP | PvP有効 / 無効、友軍誤射設定 |
| 経験値 | 経験値倍率 |

## Shifting Sands DLCのマルチプレイ

DLCのエジプトマップとメインマップ（Cloud Mist Forest）は**別のワールド**として機能します。

- **シングルプレイ / Co-op**: マップ間ポータルは使用不可。スタートアップメニューからどちらのマップをプレイするか選択
- **キャラクター引継ぎ**: Shifting Sandsマップ開始時に、本編のキャラクターをスキル込みでインポート可能（アイテムは持ち込み不可）
- **専用サーバー（マルチ）**: マップ下部のCross-Server Portalを使ってマップ間を移動可能
