# マルチプレイ

> 📌 情報源: [Steam ストアページ](https://store.steampowered.com/app/2646460/Soulmask/)

## マルチプレイの種類

| モード | 概要 |
|--------|------|
| オンラインCo-op | インターネット経由での協力プレイ |
| LANプレイ | 同一ネットワーク内での協力プレイ |
| 公式サーバー | 公式が提供するサーバーへの参加 |
| 専用サーバー | プレイヤーが独自に立てるサーバー |

## 最大人数

- **Co-op**: 最大人数は情報収集中（公式サーバーでは多人数対応）
- **カスタマイズ**: 200以上のサーバー設定項目でゲームバランスを調整可能

## マルチプレイの特徴

### 協力プレイの利点
- 部族員の確保・拠点建築が効率化
- ボス攻略の難易度が大幅に低下
- 役割分担（戦闘特化・クラフト特化など）が可能

### PvP要素
Shifting Sands DLCではエアシップを使ったPvP空中戦闘が追加されます（[エアシップ](shifting-sands/airship.md)参照）。

サーバー設定によってPvPのオン/オフを選択できます。

## サーバーの設定

200以上のカスタマイズ項目により、以下のようなゲームバランスを調整可能です：

- 素材の採集量・ドロップ率
- 部族管理の難易度
- PvP / PvEの切り替え
- デイサイクルの速度
- その他多数（詳細は情報収集中）
