## DLC追加コンテンツ

### 新マップ: エジプトマップ
砂漠・ナイル川地帯を舞台にした広大な新フィールド。オアシス要塞・嵐帯・ダンジョン・移動回廊など多彩なロケーションが存在します。

→ 詳細: [エジプトマップ](egypt-map.md)

### エアシップ（空中要塞）
木製スキフから反重力エンジン搭載の上位機種まで段階的に進化するエアシップ。空中基地として機能し、PvP/PvE空中戦闘にも対応。

→ 詳細: [エアシップ（空中要塞）](airship.md)

### 神マスク（4種）
エジプトの神々をモチーフにした4種の新マスク。飛行・浸食・防御・太陽光線など固有の能力を持ちます。

→ 詳細: [神マスク](god-masks.md)

### 新敵・新ボス
エジプトマップ固有の新敵が多数登場。フェーズパターン・マスク相性・エアシップ搭載オプションを組み合わせた攻略が必要になるボスも。

### 空中戦闘システム
プレイヤー対プレイヤー（PvP）・PvEの空中戦闘が可能に。エアシップに武器を搭載して空中戦を展開します。

## 本体との連携

DLCのエジプトマップは本体マップとは別のワールドとして機能します（接続方法は情報収集中）。部族員・資源・クラフトレシピは一部共有される可能性があります（詳細確認中）。
