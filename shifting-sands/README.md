## DLC追加コンテンツ

### 新マップ: エジプトマップ
砂漠・ナイル川地帯を舞台とした広大な新フィールド。オアシス要塞・嵐帯・ダンジョン・移動回廊など多彩なロケーション。

→ 詳細: [エジプトマップ](egypt-map.md)

### エアシップ（空中要塞）
木製スキフから反重力エンジン搭載の上位機種まで段階的に進化するエアシップ。空中基地・PvP/PvE空中戦闘に対応。

→ 詳細: [エアシップ（空中要塞）](airship.md)

### 神マスク（4種）
エジプトの神々をモチーフにした4種の新マスク。ホルス（飛行）・アヌビス（アンデッド召喚）・ソベク（水中・防御）・アムン・ラ（太陽エネルギー）の固有能力を持つ。

→ 詳細: [神マスク](god-masks.md)

### 新マウント（3種）
ロバ・ラクダ・サイの3種が追加。ラクダは砂漠の長距離移動、サイは戦闘に特化。

→ 詳細: [騎乗・ペット](../mounts-pets.md)

### 空中戦闘システム
エアシップに武器を搭載してのPvP / PvE空中戦。重火器で特定ボスのシールドフェーズを無効化できる。

## マップへのアクセス方法

DLCのエジプトマップ（Shifting Sands）と本編マップ（Cloud Mist Forest）は**別のワールド**として機能します。

| プレイスタイル | アクセス方法 |
|-------------|------------|
| **シングルプレイ / Co-op** | スタートアップメニューから使用するマップを選択。本編キャラクターのスキルをインポート可能（アイテムは持ち込み不可） |
| **専用サーバー（マルチ）** | マップ左下のCross-Server Portalを使ってマップ間を移動可能 |

> キャラクターインポートを使うと、本編で育てたキャラクターのスキルを引き継いでDLCマップを始めることができます。
