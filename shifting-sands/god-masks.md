# 神マスク

> 📌 情報源: [fextralife.com](https://fextralife.com/soulmask-all-new-masks-in-shifting-sands-and-how-to-get-them/) / [games.gg](https://games.gg/soulmask/guides/soulmask-shifting-sands-god-masks-and-how-to-get-them/) / [bisecthosting.com](https://www.bisecthosting.com/blog/soulmask-shifting-sands-masks-guide-locations-new-effects)

Shifting Sands DLCでは、エジプトの神々をテーマにした**4種の神マスク**が追加されます。各マスクは固有の能力と専用ボスから入手する仕組みです。

---

## ホルス（Horus）: Celestial Wings

**飛行・探索特化型**

| 項目 | 内容 |
|------|------|
| テーマ | 空の神・太陽神ホルス |
| 入手方法 | **DLC開始直後にイントロシーケンスで自動入手**（ボス不要） |
| 推奨スタイル | マップ探索・スカウト・高所からの奇襲 |

**主要アビリティ:**
- **飛行（Flight）**: ミミクリーモード発動中に空中を自由に飛行できる
- 敵陣を上空からスカウトして安全なルートを確認
- 難所の地形を空中から一気に突破
- 戦闘中の空中機動で攻防の選択肢を増やす

> ホルスはDLCの最初に手に入るマスクで、エジプトマップの広大な砂漠を移動するのに不可欠です。

---

## アヌビス（Anubis）: Doomsday Judgment

**アンデッド軍・状態異常特化型**

| 項目 | 内容 |
|------|------|
| テーマ | 死の神・来世の守護者アヌビス |
| 入手場所 | Sunken City（沈没都市）エリア |
| 入手方法 | **Anubis Hound（アヌビスの猟犬）**ボスを撃破 → 内部の祠でマスク入手 |
| アクセス | Mysterious Portal（謎のポータル）経由で北の古代ピラミッドへ |
| 推奨スタイル | 積極的な戦闘・ボスの長期戦・密集した敵集団 |

**主要アビリティ:**
- **状態効果反射（Status Reflection）**: ミミクリーモード中に受けた状態異常を攻撃者に反射
- **アンデッド召喚（Undead Army）**: 倒した敵を一時的に復活させて味方として戦わせる
- **包帯強化（Healing Boost）**: 包帯の持続時間+50%

---

## ソベク（Sobek）: Abyss Overlord

**水中・防御・クリティカル特化型**

| 項目 | 内容 |
|------|------|
| テーマ | ワニの神・水と力の象徴ソベク |
| 入手場所 | Three River Junction（三川合流点）/ Barren Sandland |
| 入手方法 | **Giant Crocodile（巨大ワニ）**ボスを撃破 → ピラミッド内でマスク入手 |
| アクセス | Barren Sandland Mysterious Portal付近 |
| 推奨スタイル | 水中探索・遠距離攻撃対策・クリティカルビルド |

**主要アビリティ:**
- **水中マスタリー（Aquatic Mastery）**: 酸素量増加 + 水中移動速度アップ
- **矢ダメージ軽減（Ranged Defense）**: 受ける弓矢ダメージをパッシブで軽減
- **クリティカルバフ**: クリティカル率+20% + 自己回復能力強化

---

## アムン・ラ（Amun-Ra）: Glorious Hymn

**太陽エネルギー・弱点露出型**

| 項目 | 内容 |
|------|------|
| テーマ | 太陽神アムン・ラ |
| 入手場所 | Rock Lake（岩湖）北部の Ancient Pyramid |
| 入手方法 | **Scarab（スカラベ）**ボスを撃破 → ピラミッド内でマスク入手 |
| アクセス | Rock Lake Mysterious Portal経由 |
| 推奨スタイル | 範囲攻撃・弱点を突く戦闘スタイル |

**主要アビリティ:**
- **太陽エネルギー（Solar Energy）**: ミミクリーモード中、周囲の敵に継続的なダメージパルスを与える
- **弱点露出（Weak Point Exposure）**: レーザーエネルギーで敵の弱点を露出させ、致命的な攻撃を通しやすくする
- **焼却（Incineration）**: 強力な燃焼（DoT）ダメージを付与

## 推奨マスク選択

| 目的 | 推奨マスク |
|------|-----------|
| DLC開始直後の探索 | **ホルス**（自動入手・飛行で砂漠を制覇） |
| ボス攻略・アンデッド戦略 | **アヌビス** |
| 水中探索・矢対策 | **ソベク** |
| 広範囲への継続ダメージ | **アムン・ラ** |

## 神マスクの入手順序（推奨）

1. **ホルス** → DLC開始時に自動入手
2. **アヌビス** → Sunken Cityへ向かい、Anubis Houndを撃破
3. **ソベク** → Three River Junctionの巨大ワニを撃破
4. **アムン・ラ** → Rock Lake北部のScarabを撃破

各ボスへのアクセスはMysterious Portal（謎のポータル）を経由します。ポータルへの到達はエアシップ（[エアシップ](airship.md)参照）を使うと効率的です。
