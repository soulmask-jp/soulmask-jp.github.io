# 神マスク

> 📌 情報源: [soulmaskshiftingsands.wiki](https://soulmaskshiftingsands.wiki/)

Shifting Sands DLCでは、エジプトの神々をテーマにした**4種の神マスク**が追加されます。各マスクは固有の能力を持ち、探索・戦闘・防御などプレイスタイルに応じた選択が可能です。

## ホルス（Horus）

**飛行・探索特化型**

| 項目 | 内容 |
|------|------|
| テーマ | 空の神・太陽神ホルス |
| 主能力 | 飛行能力・滑空制御 |
| 特性 | 移動・探索を大幅に加速 |
| 推奨スタイル | 広大なエジプトマップの探索、マップ全域のスカウト |

**主要アビリティ（詳細は情報収集中）**:
- 飛行モードへの切り替え
- 滑空による高速移動
- 高所からの地形把握

## アヌビス（Anubis）

**戦闘・攻撃特化型**

| 項目 | 内容 |
|------|------|
| テーマ | 死の神・来世の守護者アヌビス |
| 主能力 | 浸食ダメージ・敵復活 |
| 特性 | 戦闘テンポを支配 |
| 推奨スタイル | 積極的な戦闘、ボス攻略 |

**主要アビリティ（詳細は情報収集中）**:
- 浸食（Erosion）ダメージを付与
- 倒した敵を一時的に復活させて戦闘に参加させる
- 戦闘のペースを掌握するアビリティ

## ソベク（Sobek）

**防御・タンク型**

| 項目 | 内容 |
|------|------|
| テーマ | ワニの神・水と力の象徴ソベク |
| 主能力 | 水盾・トラップ無効 |
| 特性 | 防御力を大幅に強化 |
| 推奨スタイル | 拠点防衛、困難な敵との正面戦闘 |

**主要アビリティ（詳細は情報収集中）**:
- 水のシールドで受けるダメージを軽減
- トラップやデバフを無効化
- 高い防御力で敵の攻撃を受け続ける

## アムン・ラ（Amun-Ra）

**バランス・攻防型**

| 項目 | 内容 |
|------|------|
| テーマ | 太陽神・万能の神アムン・ラ |
| 主能力 | 太陽光線・昼間HP再生 |
| 特性 | 瞬発力と持続力を両立 |
| 推奨スタイル | 汎用的な戦闘スタイル |

**主要アビリティ（詳細は情報収集中）**:
- 太陽光線による遠距離攻撃
- 昼間のHP自動再生
- 攻守のバランスが取れた能力

## マスク選択の指針

| 目的 | 推奨マスク |
|------|-----------|
| マップ探索を効率化したい | **ホルス** |
| 戦闘を積極的に楽しみたい | **アヌビス** |
| 敵の攻撃に耐えたい・防衛重視 | **ソベク** |
| 攻守バランスよく動きたい | **アムン・ラ** |

## 神マスクの入手方法

情報収集中。エジプトマップのボス攻略・特定ダンジョンクリアで入手できると予想されます。

## 本編マスクとの比較

本編の5種マスクとDLCの4種神マスクは用途が異なります。DLC環境での探索・戦闘には神マスクが適している場合が多いです（本編マスクの詳細は[マスクシステム](../mask-system.md)参照）。
