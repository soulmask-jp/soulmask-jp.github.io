# エジプトマップ

> 📌 情報源: [soulmaskshiftingsands.wiki](https://soulmaskshiftingsands.wiki/) / [PC Gamer](https://www.pcgamer.com/games/survival-crafting/soulmask-egyptian-massive-dlc-shifting-sands-base-building-quuland-games/)

## マップ概要

Shifting Sands DLCのエジプトマップは、**本体と同サイズ**の砂漠・ナイル川地帯を舞台とした広大なマップです。

## 主要エリア

| エリア | 特徴 |
|--------|------|
| **オアシス要塞（Oasis Fortress）** | 水源に守られた砂漠の中核拠点。資源・蛮族が集中。序盤の安全ルート確立に重要 |
| **嵐帯（Storm Zone）** | 強力な敵が生息する高難度エリア。リスクは高いが報酬も高い |
| **Sunken City（沈没都市）** | アヌビスマスクを入手できるエリア。ダンジョン探索の目玉 |
| **Three River Junction（三川合流点）** | ソベクマスクのボス（巨大ワニ）が出現するエリア |
| **Rock Lake（岩湖）北部** | アムン・ラマスクのボス（スカラベ）が出現する古代ピラミッド |
| **Mysterious Portal（謎のポータル）** | 各ボスエリアへの入口。マップ各地に配置 |
| **ナイル川流域** | 農業・水資源が豊富。農業ラインを構築するのに最適な場所 |
| **Barren Sandland（不毛の砂地）** | 過酷な砂漠環境。上位素材が採集できる高難度エリア |

## 砂漠環境の特性

| 要素 | 詳細 |
|------|------|
| **砂嵐（Sandstorm）** | 定期的に発生。視界が大幅に低下し、移動が困難になる。エアシップの補強や室内待機で対応 |
| **熱中症ダメージ** | 炎天下での活動は体温・水分消費が増加する可能性あり（詳細確認中） |
| **夜間イベント** | 夜になると特殊な敵が活動を開始する。拠点の防衛ラインを整備しておくこと |

## 資源・素材

エジプトマップには本編にはない固有の素材が分布しています。

- 神マスク強化に必要な素材（Power Crystal等）
- オアシス周辺の農業作物（本編と一部共通）
- ナイル川流域の特殊植物・鉱石
- 砂漠固有の建築・クラフト素材

## 探索のコツ

**ホルスマスクで始める**  
DLC開始時に自動入手できるホルスマスクの飛行能力で、まず安全なオアシスエリアを見つけて拠点を確立しましょう。

**エアシップで移動効率を上げる**  
砂漠マップはエアシップでの移動が最も効率的です（[エアシップ](airship.md)参照）。Tribe Boatを早めに建造して砂漠の移動手段を確保しましょう。

**オアシスを拠点にする**  
オアシス要塞周辺は水・食料・蛮族が揃っており、最初の安定した拠点候補として最適です。

**嵐帯には装備を整えてから**  
嵐帯は高難度エリアです。十分な装備・部族員・エアシップ武装を整えてから挑みましょう。
