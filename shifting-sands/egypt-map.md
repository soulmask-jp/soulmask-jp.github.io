# エジプトマップ

> 📌 情報源: [soulmaskshiftingsands.wiki](https://soulmaskshiftingsands.wiki/) / [PC Gamer](https://www.pcgamer.com/games/survival-crafting/soulmask-egyptian-massive-dlc-shifting-sands-base-building-quuland-games/)

## マップ概要

Shifting Sands DLCのエジプトマップは、**本体と同サイズ**の砂漠・ナイル川地帯を舞台とした広大なマップです。

## 主要エリア

| エリア | 特徴 |
|--------|------|
| オアシス要塞 | 水源に守られた砂漠の拠点。資源と蛮族が集中 |
| 嵐帯（Storm Zone） | 強力な敵が生息。高リスク高リターンのエリア |
| ダンジョン入口 | 各地に散在。内部には新ボスと報酬 |
| 移動回廊 | マップ各地を繋ぐ主要通路 |
| ナイル川流域 | 農業・水資源が豊富。農業ラインに最適 |

## 砂漠環境の特徴

砂漠マップならではの環境要素があります（詳細は情報収集中）：

- **熱中症・乾燥ダメージ**: 水分・体温管理が重要
- **砂嵐イベント**: 一定周期で砂嵐が発生し、視界が悪化
- **夜間イベント**: 夜になると特殊な敵が活動開始

## 資源配置

砂漠マップ固有の素材・資源が分布しています（具体的な素材名は情報収集中）：

- エジプト固有の鉱石・素材（神マスク強化素材等）
- オアシス周辺の農業資源
- ナイル川流域の特殊素材

## 探索のコツ

**エアシップを活用する**  
砂漠マップはエアシップでの移動が効率的です。高所からマップ全体を把握してから探索するのが有効です（[エアシップ](airship.md)参照）。

**水分管理を優先する**  
砂漠環境では水分消費が激しい可能性があります。オアシス・ナイル川を補給ポイントとして活用しましょう（詳細は情報収集中）。

**夜間イベントへの備え**  
夜間に活動する強力な敵に備えて、拠点の防衛ラインを整備しておきましょう。
