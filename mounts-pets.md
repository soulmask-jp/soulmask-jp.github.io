# 騎乗・ペット

> 📌 情報源: [gamerant.com](https://gamerant.com/soulmask-all-mounts-how-to-get-them/) / [games.gg](https://games.gg/soulmask/guides/soulmask-shifting-sands-mounts-guide/) / [gameskinny.com](https://www.gameskinny.com/tips/soulmask-best-mounts-animals-and-tames-locations/)

## マウントの基本

特定の動物を幼体から育てることで騎乗可能なマウントになります。**成体は飼いならせません**。幼体または卵を入手して自分で育てる必要があります。

### 共通の飼育手順
1. 幼体（Baby）を罠で捕獲、または卵を入手
2. 食料・水を与えながら成体まで育てる
3. **サドル（Saddle）**を装備して騎乗可能に

## 本編マウント一覧（5種）

| マウント | 必要認識力 | 入手方法 | 特性・用途 |
|---------|----------|---------|-----------|
| **アルパカ（Alpaca）** | 15 | 中サイズの罠で幼体を捕獲 | 序盤の基本マウント。積載量350。好物：わら・ジャガイモ・キヌア・カボチャ・トウモロコシ・バナナ・パイナップル・ピーナッツ |
| **ラマ（Llama）** | 15 | アルパカと同様の方法で捕獲 | アルパカより**積載量が大幅に優れる（800）**。好物：パイナップル・ピーナッツ・ラマ専用エサ |
| **ジャガー / パンサー（Jaguar / Panther）** | 25 | 青い足跡を追って幼体を罠で捕獲 | **高速移動に最適**。ジャガーは斑点模様、パンサーは黒毛で同系種 |
| **ダチョウ（Ostrich）** | 36 | 卵を見つけて熱源付近で孵化させる | 独特の**グライド能力**を持つ。中距離移動に便利 |
| **ユキヒョウ（Snow Leopard）** | 51 | ジャガーと同様の方法で捕獲（北西の雪山エリア） | **最高レベルのエンドゲームマウント**。入手は難しいが性能最高 |

### マウント選択の指針

| 目的 | 推奨マウント |
|------|------------|
| 序盤の移動 | **アルパカ**（入手しやすい） |
| 荷物運搬 | **ラマ**（積載量800で圧倒的） |
| 高速移動・戦闘 | **ジャガー / パンサー** |
| 探索・グライド | **ダチョウ** |
| 後半の最強マウント | **ユキヒョウ** |

## Shifting Sands DLC マウント（3種）

エジプトマップ追加のDLCで3種の新マウントが解放されます。

| マウント | 特性 | 捕獲方法 |
|---------|------|---------|
| **ロバ（Donkey）** | 最も入手しやすい。初期段階の運搬用。マップ全域で見つかる | 幼体を手で拾って連れ帰る |
| **ラクダ（Camel）** | ロバより速度・積載量が優秀。**砂漠での長距離移動に最適** | 中サイズの罠（Medium Trap）で幼体を捕獲 |
| **サイ（Rhino）** | 最高の攻撃力と基本HPを持つ**戦闘向きマウント**。鞭で速度ブースト可能 | 幼体を罠で捕獲 |

## ペットシステム

マウント以外の動物もペットとして連れて歩いたり、拠点で飼育できます。

### 主な飼育動物と用途

| 動物 | 飼育施設 | 主な産出品 |
|------|---------|----------|
| カピバラ（Capybara） | カピバラ・ペン | 素材・肉 |
| シチメンチョウ（Turkey） | ターキー・コープ | 肉・羽 |
| ピラニア（Piranha） | ピラニア・プール | 肉・魚素材 |
| アルパカ / ラマ | バーン（Barn） | 乗用・運搬 |

### 繁殖農場（Breeding Farm）
素材：粘土20・青銅インゴット60・木材100・革ロープ20  
自前の繁殖農場を建てることで、フィールドへ出なくても飼育動物を補充できます。
